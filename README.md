# Kanban Board

A responsive Kanban task management application built with React.

## Live Demo
🔗 [https://kanban-board-navy-three.vercel.app/](https://kanban-board-navy-three.vercel.app/)

## Features
- Create tasks with title and description
- Drag and drop tasks between columns (Todo, In Progress, Done)
- Edit and delete tasks
- Task counter per column
- Data persists after page refresh (localStorage)
- Responsive design for mobile and desktop

## Tech Stack
- React (Vite)
- @hello-pangea/dnd (drag and drop)
- CSS
- Deployed on Vercel

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/AkshatMaheshwari27/kanban-board.git
cd kanban-board
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```

App will start at `http://localhost:5173`
